# 简介

    项目结构
    main 源代码文件
        -bankboot
            -config 配置文件
            -controller 控制器
            -dao 数据访问层
            -domain 实体类型
            -server 服务层
            application 服务器启动主类
        -resources 资源文件夹
            -static 静态可访问资源(适合放置网页前端界面资源)
            application.yml 项目配置文件
    test 测试文件
        -java
            -bankboot
                -dao 数据层接口测试
                -server 服务层接口测试
